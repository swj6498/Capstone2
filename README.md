## 📖 프로젝트 개요

팟토피아는 **게시물 작성, 채팅, 위치** 기능을 제공하는  
커뮤니티 앱입니다.

- 개발 기간 : `2025.03.17 ~ 2025.05.27`
- 개발 인원 : `4명`

---

### 👨‍💻 담당 역할

- FlutterFire 환경 구축
- 프론트엔드 구현
- 구글 지도 API 연동
- 거리 계산 기능 구현

### ✨ 주요 특징

  - 🔍 **Firestore 조건 쿼리 검색 + 해시태그**
    → 고급 필터링 및 해시태그 기반 게시물 검색​

  - 💬 **실시간 채팅방 (Firestore 동기화)**
    → 채팅방 생성/참가자 구조 + 메시지 실시간 동기화​

  - 👤 **Firebase Auth + 인증코드 검증**
    → 회원가입 시 SMS/이메일 인증코드 발송 및 검증

  - 📍 **Google Maps API + Geocoding 연동**
    → 게시물 거리순 정렬 + 위치 기반 검색

  - 🤖 **Google Gemini 기반 채팅 비속어 필터링 기능**  
    → 사용자 채팅의 은어/비속어 필터링
