## 📖 프로젝트 개요

팟토피아는 **게시물 작성, 채팅, 위치** 기능을 제공하는  
커뮤니티 앱입니다.

- 개발 기간 : `2025.03.17 ~ 2025.05.27`
- 개발 인원 : `4명`

---

### 👨‍💻 담당 역할

- 파이어베이스 플러터 연동
- 프론트엔드 구현
- 구글 지도 API 연동
- 거리 계산 기능 구현
